# 🎮 Hangman Elite (Streamlit Game)

## 📌 Overview

This project is a text-based Hangman game developed using Python and Streamlit.
In this game, we try to guess a hidden word by selecting letters one by one.
If the guessed letter is correct, it appears in the word.
If the guessed letter is wrong, the number of remaining attempts decreases.

The game continues until we either guess the complete word or lose all lives.

---

## 🚀 Features

* Multiple categories like Programming, Design, and Pakistan
* Interactive buttons for letter selection
* Lives system using heart symbols
* Win and lose conditions
* Restart game option
* Progress bar showing survival chances
* Simple and attractive user interface using CSS

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Random module

---

## ▶️ How to Run the Project

Step 1: Install Streamlit
pip install streamlit

Step 2: Run the project
streamlit run app.py

Step 3: Open the browser and play the game

---

## 🎮 How to Play

1. Select a category from the sidebar
2. Click on New Game
3. Guess letters by clicking buttons
4. Each wrong guess reduces one life
5. Guess the complete word before lives reach zero
