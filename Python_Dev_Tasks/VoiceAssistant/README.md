# 🎤 Voice Assistant using Python

## 📌 Project Overview

This project is a **custom voice assistant built using Python** that can recognize voice commands, perform automated tasks, and respond using speech. It is designed to simplify daily tasks through voice interaction and automation, similar to virtual assistants like Siri or Google Assistant.

---

## 🚀 Features

* 🎤 Voice Recognition (Speech to Text)
* 🔊 Text to Speech Response
* 🕒 Tells Time and Date
* 🌐 Opens Websites (Google, YouTube)
* 📚 Wikipedia Search (Who is / What is)
* 🎵 Play Songs on YouTube
* 😂 Jokes Generator
* 🧠 Study Mode Activation
* 🖥️ GUI Interface using Tkinter
* ⚙️ Modular Code Structure

---

## 🛠️ Technologies Used

* Python 3.13
* SpeechRecognition
* pyttsx3 (Text to Speech)
* pywhatkit
* wikipedia
* tkinter (GUI)
* webbrowser
* datetime

### Run GUI version:

```bash

```


## 🎯 How It Works

1. The assistant listens to your voice input 🎤
2. Converts speech into text
3. Matches command with predefined functions
4. Executes task (open app, search, play music, etc.)
5. Responds back using voice 🔊
